为了将人类赶出查尔，虫群必须冲垮无畏高地堡垒， 而接近堡垒的唯一办法就是通过亡骨壕沟 戈尔贡级战列巡航舰随时可能从天而降。

# 10. 永远消失的信使

星灵仍在想方设法警告萨古拉斯这里的情况， 他们企图发射穿梭机穿过折跃网络前往萨古拉斯。 阻止任何穿梭机进入航道！

# 13. 唤醒虫祖

泽鲁斯，异虫发源地。远古的意识体正在此沉睡。 虫祖，这只原始异虫亲眼见证了主宰的诞生。 为寻找泽鲁斯真正的力量，凯瑞甘需要唤醒它。

# 14. 破茧重生

泽鲁斯这个异虫诞生之地充满着无尽严峻考验。 这最古老分裂池，一个精华演变出了最早的异虫。 在这里，将能得到力量，进化，改造，升华。

# 15. 至尊霸主

是时候迎击剩余的原始族群首领了： 亚格卓拉、凯雷斯、斯利凡以及…… 将他们的都占为己有，拥抱所有异虫的命运。

# 17. 感染危机

从科普卢星区接收到了来自斯托科夫的灵能信号， 蒙斯克在此斯基格尔研究站研究培育混合体； 击溃他们的防御，进入研究站阻止蒙斯克的研究。

# 18. 幕后真凶

混合体由异虫和星灵的 DNA 混合培养而成。 埃蒙仆从，古老的生物纳鲁德博士是这里的主人。 虫群已经准备就绪，是时候杀进去一探究竟了。

# 19. 虚空幻影

纳鲁德是一个古老的生物，拥有强大的力量。 他将塔达林星灵招至麾下，作为王牌部队。 现在，他正在实验室的腹地等待着，杀了他！

# 22. 罪证确凿

情报显示雷诺被关押在摩洛斯号机动性监狱船上， 这艘舰船将在阿特拉斯空间站短暂停靠进行补给。 从帝国手中救回吉姆・雷诺。

# 24. 天降虫海

虫群部队已经在克哈上空集结，准备进攻克哈。 克哈轨道防御系统将对降落前的虫群造成巨大损失。 地面入侵克哈，使用胆汁喷射体摧毁轨道防御系统。

# 26. 兵临城下

虫群已经集结完毕，奥古斯特格勒战役即将打响。 而人类部署了灵能粉碎器，企图腐蚀虫群主巢心智， 凯瑞甘派遣德哈卡去摧毁它，防止它从内部瓦解虫群。

# 27. 最终审判

人类的终极武器已被摧毁，虫群大军蓄势待发。 整个克哈，现在只剩下蒙斯克王宫还在负隅顽抗。 所有的虫母、虫后们，撕碎他们，不要停下，进攻！
