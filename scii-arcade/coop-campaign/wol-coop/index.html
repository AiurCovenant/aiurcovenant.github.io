# 简介

在《星际争霸 II》八周年，和你的好友一同再次游玩自由之翼战役

  • 三人合作游戏。
  • 三人共享支援。
  • 基于扩展来支持最多三名玩家。
  • 4 种难度可选。
  • 科技、雇佣兵、研究可存档。
  • 把敌人打个落花流水！

与你的好友一同再次游玩自由之翼战役。

本系列战役由 LordDz 制作，由 MengLuoRJ 负责简中本地化维护，由 Siyim 协助简中翻译。

提示

# 1. 自由曰 1. 自由日

因关键字 “日” 被简体中文区域网易暴雪定义为不可接受，故本关采用名称 “自由曰”。

  • 链接：battlenet:://starcraft/map/5/81472

# 2. 不法之徒

帝国军正在玛・萨拉挖掘一个外星神器，夺取这个神器可以沉重打击帝国。

  • 链接：battlenet:://starcraft/map/5/81473

# 3. 零点行动

  • 链接：battlenet:://starcraft/map/5/81494

# 4. 来之不易

莫比斯基金会认为有另一件神器在 “蒙莱斯” 星球上，他们通过泰凯斯雇佣雷诺的游骑兵们去夺取它。

  • 链接：battlenet:://starcraft/map/5/81519

# 5. 大撤离

收到此求救信号的舰队请注意…… 虫群正在入侵阿格瑞亚…… 帝国军抛弃了我们…… 我们需要立刻撤离，请帮帮我们！

  • 链接：battlenet:://starcraft/map/5/81533

# 6. 大爆发

当阿格瑞亚的难民逃到梅茵霍夫时，这里密集的营区已蔓延着异虫感染病毒。汉森博士再次希望雷诺阻止这一切。

  • 链接：battlenet:://starcraft/map/5/81537

# 7. 火车大劫案

有消息声称帝国正在塔桑尼斯执行新的回收行动，许多补给火车几乎没有防御。如果能在他们离开前成功拦截火车，雷诺的游骑兵们将能够极大获益。

  • 链接：battlenet:://starcraft/map/5/81548

# 8. 博弈

雷诺的雇佣兵们在亡者之港雇佣了奥尔兰上校，打算破解他们从塔桑尼斯找到的副官。但是，奥尔兰正打算把副官卖给帝国。

  • 链接：battlenet:://starcraft/map/5/81576

# 9. 恶魔游乐场

托什，一个反帝国的幽魂，雇佣了雷诺前往红石三星球上发笔横财。但那喷涌的熔岩和入侵的虫群这不是一次常规的采矿行动。

  • 链接：battlenet:://starcraft/map/5/81606

# 10. 挖宝行动

莫比斯基金会在席尔上的一座萨尔纳加神庙中发现了一块碎片，但派去的探险小队神秘失踪。现在，他们雇佣雷诺的游骑兵去调查并完成任务。

  • 链接：battlenet:://starcraft/map/5/81617

# 11. 末曰密语 1. 末日密语

因关键字 “日” 被简体中文区域网易暴雪定义为不可接受，故本关采用名称 “末曰密语”。

泽拉图得知了一个预示着萨尔纳加回归的预言。他与他的学徒们正前往乌纳去探寻这个预言。

  • 链接：battlenet:://starcraft/map/5/81665

# 12. 恶兆

泽拉图在乌兰找到了一个预言碎片，现在他与盟友们一路来到了文献禁地，扎库尔，去寻求解释。

  • 链接：battlenet:://starcraft/map/5/81666

# 13. 末曰回响 13. 末日回响

在乌兰发现的预言中提到了主宰，泽拉图回到了已成为废墟的艾尔，去造访那主宰已经干瘪的躯体，希望能从他们卷须中获得主宰相关的记忆。

  • 链接：battlenet:://starcraft/map/5/81667

# 14. 究极黑暗

泽拉图通过主宰的记忆看到了未来的幻象 —— 刀锋女王已经倒下，泰伦已被吞噬，虫群落入了埃蒙之手，而星灵正团结在一起与埃蒙和他的混合体进行最后的生死之战。

  • 链接：battlenet:://starcraft/map/5/81670

# 15. 莫比斯代理人

刀锋女王已经发现是莫比斯基金会资助并雇佣雷诺去获取那些神器，她正在攻击泰拉多尔的基金会研究园区。保护神器研究资料，疏散转移莫比斯基金会。

  • 链接：battlenet:://starcraft/map/5/81714

# 16. 欢迎来到丛林

托什再次雇佣游骑兵们去做点脏活。在拜舍尔有一种稀有瓦斯，我们称他 “地嗪”，那玩意儿价值连城。而星灵称其为 “造物主之息”。

  • 链接：battlenet:://starcraft/map/5/81717

# 17A. 营救

任务 17 需要选择 A 或 B 任务游玩。

帮助托什进入新福尔松监狱，释放那些被囚禁的幽魂、逃犯以及小提琴音乐家。而托什会留在您的身边，帮助您训练幽魂。

  • 链接：battlenet:://starcraft/map/5/81748

# 17B. 幽灵一击

任务 17 需要选择 A 或 B 任务游玩。

托什简直就是个疯子，他不顾任何平民的安全也要执意训练幽魂并将他们不断袭击帝国。帮助诺娃摧毁位于阿佛纳斯的幽魂训练设施，消除一切可能的恐怖威胁。而诺娃会教会您如何训练幽灵，但不会加入游骑兵。

  • 链接：battlenet:://starcraft/map/5/81772

# 18A. 拯救海文

任务 18 需要选择 A 或 B 任务游玩。

撤离梅茵霍夫的殖民者前往了海文，但他们仍被异虫病毒感染。而这颗靠近星灵空间的星球被执行官塞兰迪斯发现了其中的感染，而她正带领着一支舰队前去净化海文。与人类难民并肩作战，无论最后汉森博士是否找到治愈感染的办法，这都将促进对异虫的研究。

  • 链接：battlenet:://starcraft/map/5/81802

# 18B. 海文的陷落

任务 18 需要选择 A 或 B 任务游玩。

汉森博士为那些撤离梅茵霍夫的殖民者找到了新的殖民星球 —— 海文，但大批殖民者仍感染了异虫孢子。现在，游骑兵们正式介入清理感染。与昔日盟友 —— 星灵，共同作战。这虽与汉森的方案相左，但这能彻底净化这颗星球，解决问题。而星灵也答应愿意分享一些技术和知识。

  • 链接：battlenet:://starcraft/map/5/81803

# 19. 超新星

逃离虫群入侵的莫比斯基金会重整旗鼓，他们找到了神器的四块碎片 —— 位于一个名为提丰的小星球上。恒星就要爆炸，任务很简单，找到神器 —— 然后别被烧死。

  • 链接：battlenet:://starcraft/map/5/81814

# 20. 虚空巨口

在从提丰取得神器碎片后，雷诺发现莫比斯基金会属于帝国王子瓦伦里安・蒙斯克，而瓦伦里安找到了最后一块碎片的下落 —— 在一艘废弃的萨尔纳加飞船上。

  • 链接：battlenet:://starcraft/map/5/81826

# 21. 毁灭引擎

破译后的 “联邦副官” 存载着蒙斯克当年下令屠戮塔桑尼亚的罪证，游骑兵们打算将他公之于众。帝国很快就将在克哈公布名为 “奥丁” 的新式战争机器，劫持它，就能用它攻入 UNN 工作室，在那里公布蒙斯克的滔天罪行。

  • 链接：battlenet:://starcraft/map/5/81879

# 22. 媒体轰炸

奥丁即将在克哈的 UNN 新闻站面向公众宣传，而这正是游骑兵们计划广播蒙斯克战争罪行铁证的最好地方。泰凯斯将会有美好的一天。

  • 链接：battlenet:://starcraft/map/5/81884

# 23S. 揭露黑幕

任务 23S 是隐藏任务，游玩前需先发现隐藏秘密进行解锁。

如若玩家队伍中存在任何一人尚未解锁该隐藏任务，任务游戏将被自动取消。

在克哈找到的机密文件中提到了一个位于卡斯塔纳星球轨道平台上的高机密生物武器实验室。前往调查，看看那里到底有什么。

  • 链接：battlenet:://starcraft/map/5/81882

# 24. 地狱之门

我们的所有准备都是为了这一刻，萨尔纳加神器碎片已经集齐，瓦伦里安领导的帝国战团已经准备就绪，这将是命运的一战，进攻查尔！万事俱备，明天的查尔将不再有任何的异虫存在。

  • 链接：battlenet:://starcraft/map/5/81883

# 25A. 野兽之腹

任务 25 需要选择 A 或 B 任务游玩。

我们目前仍有足够的时间发起额外的攻势以削弱虫群的防御。尽管沃菲尔德将军建议削弱异虫的空中支援，但泰凯斯则建议率领一小支精英部队，清理查尔地下的坑道虫。一旦成功，坑道虫就不再能够在最终之战中构成威胁。

  • 链接：battlenet:://starcraft/map/5/81885

# 25B. 天崩地坼

任务 25 需要选择 A 或 B 任务游玩。

降落查尔的过程简直就是一场灾难，大量人员阵亡，虽然部队重新集结起来击退了异虫，但在组装神器前必须先解决虫群的空中部队。那些异龙和巢虫领主将轨道平台视为老巢。摧毁轨道平台，异虫的空中支援将不再能够在最终之战中构成威胁。

  • 链接：battlenet:://starcraft/map/5/81886

# 26. 背水一战

神器可能是阻止刀锋女王的关键…… 但想要成功必须付出汗水与鲜血！经历了这么多，历尽艰辛与磨难…… 我明白了一件事 —— 相互依靠才能完成这项任务。即使是付出生命…… 因为有些东西值得我们去奋斗。决战时刻，用神器化解刀锋女王的罪恶，结束这场杀戮！

  • 链接：battlenet:://starcraft/map/5/81887
