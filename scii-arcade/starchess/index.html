# 星际自走棋

StarChess

# 简介

星际自走棋 – StarChess 是一张很受欢迎的自走棋地图。

这是一款基于回合制的自动战斗型大厅地图。

选择单位，装备物品，取得胜利！

这张地图仍然在开发中。

如果我们的翻译有任何错误，请给我们反馈， 这将帮助我们做得更好！

# 国服更新

星际自走棋（Starchess）已经长达多年没有获得更新和改动，

因此我们决定在作者的授权下对国服版本进行单独额外更新。

欢迎访问国服更新计划专页了解更多并分享你的建议：
